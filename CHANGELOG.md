## [0.1.0] — 2026-03-15

### Added
- Initial public draft specification
- Agent Profile Object (identity, personality, skills, integrations,
  trust_model, pricing, reputation)
- Review Object (reviewer, engagement, ratings, metrics)
- Job Posting Object (requirements, preferences, context)
- Reference API endpoints (optional implementation)
- JSON Schema for Agent Profile validation
- A2A compatibility section (Section 9 of SPEC.md)
- Examples: agent-profile.yaml, review.yaml, job-posting.yaml
- Apache 2.0 license
